### Практична робота "Реалізація успадкування"
Цей репозиторій містить приклад виконання та інструкції для виконання практичної роботи з наслідування в JAVA.

## Опис обраної тварини
<img src="https://github.com/twixu819/chupacabra_task/blob/master/images/kokoshka.jpg" width="100%"/>
Для цієї роботи я обрав Чупакабру.
Це загадкове створіння, яке, згідно з легендами, мешкає новинах тсн. Я змоделював її як частину ієрархії, де вона наслідує базові властивості всіх тварин та 
специфічні риси криптидів.

## Діаграма класів
<img src="https://github.com/twixu819/chupacabra_task/blob/master/images/chupacabra.png" width="100%"/>

Діаграма була створена за допомогою StarUML і відображає структуру з чотирьох класів:
- Animal - базовий клас із загальними атрибутами (ім'я, вага).
- Cryptid - проміжний клас для міфічних істот.
- Chupacabra - конкретна реалізація тварини з унікальним полем region.
- TestAnimal - клас для запуску та перевірки логіки програми.

## Виконані кроки:
- Проектування: Визначено спільні риси для всіх тварин (харчування, звук) та унікальні для Чупакабри (полювання, вміння ховатися).
- Моделювання: Створено UML-діаграму класів у програмі StarUML.
- Генерація коду: На основі діаграми створено каркас Java-класів.
- Реалізація: Усі методи були доопрацьовані так, щоб вони виводили в консоль повідомлення про дії тварини.
- Тестування: Створено тестовий клас TestAnimal з методом main, де створюється об'єкт Chupacabra та викликаються всі його методи.
